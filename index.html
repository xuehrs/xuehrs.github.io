今天出去浪了，武进区大学城实在没什么好玩的，就滚回来了。走了俩小时路，真累。部分地方里面的特殊商店，比如电脑店什么的进不去，还有的关门了。还有的门面太小，还有的找不到。武进大学城大排档只有晚上开放，今天上午十点没人的。为毛不骑电频车，坐公交车什么的，因为太麻烦了，运动只有一点时间。居然还看到易买得。好像在大学新村附近呢。为毛不去恐龙园什么的，一句话，没兴趣（钱）。实际上我咸蛋习惯了。

讲个笑话：每个人都是生活大师，多数是靠已知的经验判断。不同大小的石头落地只会同时落地。就是最好的证明。

某人从高薪区转到低薪区，他的消费观念在一年之后才会得到有效的改善。如果不是本人经历，基本很难想得到这件事。高薪区生活费4000，转到底薪区，生活费还是4000.为什么，因为他看到底薪区的东西好便宜。买买买！不算账单绝对不清楚钱花在哪儿了。反过来，底薪区转高薪区呢。看到东西好贵，只会选择生活必须低标准的东西。比如放弃阳光，转到地下室。放弃客厅，选择9平米的房子。可以这么说，个人有个人的活法。

心情随笔

随个锤子，心情不好~

某人说：要求工作稳定，工资高，有房有车什么的。然后人设就崩了。

我什么都没有，只能一个人过。

我只是一个没有明天的人。

不必要多想，不必要多问。只需要照顾好自己就行。

····································································································

以上纯属骗眼泪的。

我在想，如果这一切都符合我的预期，那么我会忙得连自己想要什么都不知道。现在变了，我自己闲了下来，思考了以后该怎么办？一句话，凉拌呗。将就着不讲究，不代表我一直将就着，一直不讲究。

差不多先生常说：“凡事只要差不多就好了。何必太精明呢？”

依鄙人之见，此言差矣。人活一口气，树活一张皮。凡是差不多的都是对很多情况无从选择的自我躲避、自我催眠。

毕竟人有一口气，就是争气，什么都不争，那就是争。什么都争，那就是死得最快。最好的方式就是，你知道你需要什么，愿意用什么等价值的去换取。比如我愿意用一生颠沛流离去带你看一路的风景。比如你在，就是我的全部。你不在了，什么都无所谓了。

··········································································································

我记得我在高中上语文课的老师讲过一个题目:你的人生五样是什么?你舍弃的顺序是什么?

从那里我看到了人性的本质，后来老师告诉大家为什么没有人带纸笔。我突然明白了老师的心思。人生有许多获得和失去。也有很多拿不回的东西，我们不是超人，也没有什么特殊能力。如果能够陪伴自己一辈子，到死，唯有自己的笔和纸。

只要有笔和纸，我可以写下看到那时的一家团圆，开心的欢笑好时光。只要有笔和纸，无论在哪里，世界再怎么大，邮政也会帮忙将我的思念诉说带向远方。只要有笔和纸，我可以仗剑走天涯。

好了，不扯了，我学习了。

昨夜不小心把电脑格式化了，卧槽，我的资料…，还好我个人定了很多计划，有定期备份数据规划。

废话不讲了，直奔主题。

据说wsl可以升级wsl2，同时使用VcXsrv可以使用某个桌面。

实际上我懒得用xinit窗口，没办法，命令行流派就是好用的。遇到需要图形的就比较麻烦了。

台式机i9 10920X确实牛逼上天了。笔记本就坑了，基本整机都在价钱在2w以上，性能只有台式的一半，还有那个低压本，作用不是很大的。我想买个高u，高读写，高内存的，我选了一些参数，cpu十代i7 内存16G 硬盘1T固态x2+2t机械(总之硬盘越大越好，重要的数据保存到机械硬盘里面)。集显就够了。可惜大部分电脑都是带显卡的游戏本，没有别的笔记本符合我的需求。我想到了怎么解决电脑有Windows的问题，很简单，加个硬盘，用来装arch，随便玩玩呗。

我定了几个小目标

中等性能笔记本，服务器（高配置pc），群晖rad，电瓶车，街机定制。

后来我在想，我这么干累不累。干脆只保留了笔记本，别的钱用来投资别的地方吧。人生有诗有远方，为何不带个笔和本子去仗剑天涯呢。

我注意到某人说的B言B语，说的是专注个人说话，类似txt笔记本，但是不能评论，你想说什么就说什么。我个人认为呢，这货迟早会翻车的。但是我看到了该程序开源了，但是我不看好这软件的前途的，哈哈。

还有那个某x度盘，把pandownload作者给抓了。而且各种花式限速。我都想脱坑了。每次都要我下载都要下载官方的网盘，因为我经常格式化Windows。而且经常遇到各种资源挂了的问题。我都很生气的。我经常用pandowsload搜索mdx和mdd文件，主要是我不习惯用网络词典工具。还好我做了很多词典工具整合包。目前够用了。里面还有各种离线wiki，百科什么的。真香呢。还有某个词典能够本地发声音的，可以听各种词典的人声。蛮好用的。我会考虑找个不错的网盘工具来下载资源了。

算了，不扯了，改天在写。

关于Windows终端个人使用和评价

发表于 2020-03-27 | 更新于: 2020-03-27 | 分类于 Blog ， Windows ， 终端 |
字数统计: 384 | 阅读时长 ≈ 1

babun，此终端不错，就是基于Cygwin Terminal整合而来的。里面有各种不错的工具套装，还有zsh，可以做出很多美化。不过作者那边不怎么更新了，但是你可以使用Cygwin更新里面的东西。支持中文。可以使用终端gui套上去使用。

Cygwin Terminal，不错，有很多不错的命令行工具

git Terminal

MINGW

msys

msys2

这几个都是基于Cygwin做出来的特别终端，里面集成了很多常用工具

WSL_ArchLinux，此终端不错，主要是用来编译各种包仓库的主力机，现在只是个玩具，连自己编译aur有一定的问题。不过可以做到babun那个使用效果，之前玩了Arch好几年，说明oh-my-zsh我都玩过的。类似wsl_ubuntu等等我就不用介绍了，美化了都是差不多。

cmder，终端gui，就是ConEmu的进阶版，里面可以直接使用wsl，部分汉化，可以透明，支持壁纸

ConEmu，终端gui，里面需要进行很多设置，部分汉化，可以透明，支持壁纸

ConsoleZ，终端gui，全英文，功能单一，不可以设置透明

wsl-terminal，终端gui，部分中文，可以透明，不支持壁纸

terminus，终端gui，完美中文，里面功能要付费。但是比较坑，特别是玩主机的那个同学

fluent terminal，终端gui，windows terminal加强版

windows terminal，终端gui，应用商店里面的，微软官方出品

关于网络喷子概论

发表于 2020-03-27 | 更新于: 2020-03-27 | 分类于 Blog |
字数统计: 520 | 阅读时长 ≈ 1

​ 看到网络里面评论某人诋毁李小龙，我觉得某一部分人素质不好。后来我想了了下，就算某人再牛逼，估计见到李小龙本人也不敢随便口嗨。

​ 喷子特征就是那种，不讲道理的喷，你和他讲道理，他就不和你讲道理。你和他不讲道理，他就和你谈道理。反正节奏就在他手上。她就喜欢你那种受气的表情。好了，这事不扯了，我总结下原因，为什么喷子这么不讲道理。

​ 1.他见到李小龙的坏，所以对李小龙的一切都很讨厌。比如那些家族上世纪长辈原因呗。

​ 2.他是受害者，只是谣言所误而已。再怎么真实的故事，越传越离谱。第一人是真，第二人还是真，到了后面，某人才发现这是谣言，谁能告诉我，这故事什么时候开始不在真实了？没人说得出来。

​ 3.李小龙不是一个完美的人，有些人都是喜欢拿别人的方面跟自己比较，发现自己不如他的地方，那就挑毛病。跟一个死人杠精，也是一个人才。反正死人不会张口。

​ 4.只是喷子心情不好而已。

​ 5.没想好，以后再说吧。说不定我看多了假新闻也会变成他们这样。

​ 谣言究竟是止于智者,还是止于下一个谣言?

​ 对于上面话题，我个人评论：一件事，只要有评论，就有争议，有争议就有不符事实的事据。不符事实的事据传多了，也就变成了大家不怎么相信的谣言了。到了最后，大家都知道那个真相是谣言，后面就没人争议了。因为即使知道真相，也没用了。

讲个冷笑话，活人嘴皮子上下一合，事情就这么说定了。就算扭曲了真相，死人也不会说话。
