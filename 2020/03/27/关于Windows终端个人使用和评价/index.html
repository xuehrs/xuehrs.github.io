关于Windows终端个人使用和评价

发表于 2020-03-27 | 更新于: 2020-03-27 | 分类于 Blog ， Windows ， 终端 |
字数统计: 384 | 阅读时长 ≈ 1

babun，此终端不错，就是基于Cygwin Terminal整合而来的。里面有各种不错的工具套装，还有zsh，可以做出很多美化。不过作者那边不怎么更新了，但是你可以使用Cygwin更新里面的东西。支持中文。可以使用终端gui套上去使用。

Cygwin Terminal，不错，有很多不错的命令行工具

git Terminal

MINGW

msys

msys2

这几个都是基于Cygwin做出来的特别终端，里面集成了很多常用工具

WSL_ArchLinux，此终端不错，主要是用来编译各种包仓库的主力机，现在只是个玩具，连自己编译aur有一定的问题。不过可以做到babun那个使用效果，之前玩了Arch好几年，说明oh-my-zsh我都玩过的。类似wsl_ubuntu等等我就不用介绍了，美化了都是差不多。

cmder，终端gui，就是ConEmu的进阶版，里面可以直接使用wsl，部分汉化，可以透明，支持壁纸

ConEmu，终端gui，里面需要进行很多设置，部分汉化，可以透明，支持壁纸

ConsoleZ，终端gui，全英文，功能单一，不可以设置透明

wsl-terminal，终端gui，部分中文，可以透明，不支持壁纸

terminus，终端gui，完美中文，里面功能要付费。但是比较坑，特别是玩主机的那个同学

fluent terminal，终端gui，windows terminal加强版

windows terminal，终端gui，应用商店里面的，微软官方出品
